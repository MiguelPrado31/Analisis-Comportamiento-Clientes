# 📊 Análisis de Comportamiento de Clientes de una Empresa
Proyecto de análisis de datos enfocado en identificar los factores que influyen en el comportamiento de compra de los clientes, incluyendo descuentos, reseñas, estacionalidad y métodos de pago, utilizando Python, MySQL y Power BI para el procesamiento, análisis y visualización de los datos.

## 🧩 Planteamiento del Problema:

### *¿Cómo puede la empresa aprovechar los datos de compra de los consumidores para identificar tendencias, mejorar la participación de los clientes y optimizar las estrategias de marketing y de productos?*

## 🚀 Características:
- **Preparación de Datos y Modelado (Python)**: Limpieza y transformación del dataset para su análisis.

- **Análisis de Datos (MySQL)**: Organización y consulta de datos para analizar el comportamiento de los clientes, identificar segmentos, evaluar la lealtad y detectar los factores que influyen en las decisiones de compra.

- **Visualización de datos (Power BI)**: Creación de un panel interactivo para observar patrones y tendencias, permitiendo una visualización fácil y clara de los datos clave.
